# Screenshot Tool

一个简单但功能强大的截图工具，支持多显示器。

## 功能特点

- 多显示器支持
- 快捷键操作
- 自定义保存路径
- 简洁的用户界面

## 安装说明

1. 下载最新版本的安装包
2. 运行 setup.bat 进行安装
3. 按照提示完成安装过程

## 使用方法

1. 运行 start.bat 启动程序
2. 使用快捷键 `Ctrl + Alt + A` 开始截图
3. 选择要截取的区域
4. 截图会自动保存到指定目录

## 开发环境

- Python 3.8+
- PyQt5
- Windows 10/11
